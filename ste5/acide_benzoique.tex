\section*{Introduction}
L'acide benzoïque, de formule \ce{C6H5COOH}, et le benzoate de sodium sont des conservateurs 
antimicrobiens respectivement identifiés par les codes E210 et E211. 
Ils sont présents dans de nombreux produits alimentaires et notamment dans 
certaines boissons gazeuses sucrées.

\begin{figure}[h]
\centering
\chemfig{

}
\caption{Formule topologique de l’acide benzoïque}
\end{figure}

\begin{itemize}
    \item Masse molaire de l'acide benzoïque : $M = \SI{122}{\gram\per\mole}$.
    \item Température de fusion de l'acide benzoïque : $\theta_f = \SI{122,4}{\degreeCelsius}$.
    \item pKa du couple acide benzoïque/ion benzoate : $pKa = 4,2$.
    \item Le benzoate de sodium est soluble dans l’eau.
\end{itemize}
